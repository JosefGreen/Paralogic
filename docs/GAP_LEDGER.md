# Gap Ledger

## Top Ten Bottlenecks

1. Unified type signature and model semantics for the Paralogic / ISFT core.
2. Adeq development by domain, including coding rules and evidence thresholds.
3. Full M8 power-erasure formalization with explicit non-collapse guards.
4. Normative bridge library for harm, responsibility, repair obligation,
   accountability, legal illegitimacy, governance legitimacy, and moral guilt.
5. Repair calculus: diagnosis, plan, action, verification, partial repair,
   failed repair, repair closure, regress, and externalities.
6. Delta outcome classifier, including mutual exclusivity or intentional
   overlap rules.
7. Evaluator calculus with criteria-relative soundness and incompleteness.
8. Lean-ready minimal kernel for ISF, M8, ValidInsight, evaluator acceptance,
   and non-collapse theorems.
9. Executable finite checker expansion beyond the one-object Boolean fragment.
10. Empirical operationalization: coding manual, reliability protocol,
   validation sets, benchmarks, and falsification conditions.
