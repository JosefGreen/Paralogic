# CALM Source and Case Dossier Template

## Status

This is a template for future source work. It contains no completed case packet
and does not assert facts about any named case.

## Source Maturity Scale

| Level | Description | Allowed use |
| --- | --- | --- |
| S0 | Unsourced assertion | Not usable for factual claims |
| S1 | Personal lens or memory | Framing only, not evidence |
| S2 | Journalism or commentary | Lead, narrative context, or bounded current framing |
| S3 | Academic, technical, or professional literature | Theory, method, and conceptual support |
| S4 | Think tank, FFRDC, inspector-general summary, or official-adjacent analysis | Policy interpretation and strategic framing |
| S5 | Official report, statute, testimony, audit, budget document, or primary record | Primary factual burden |
| S6 | Multiple independent high-maturity sources in agreement | Strong support, still bounded |

## Case Packet Fields

| Field | Requirement |
| --- | --- |
| Case name | Use public name only |
| Role | Primary anchor, secondary support, human-cost case, countercase, analogy, reserve, or excluded |
| Claim supported | Exact bounded claim |
| Claim not supported | Stronger claim the case must not be used to imply |
| Source maturity | S0-S6 |
| Primary sources | Official or highest maturity sources |
| Supporting sources | Secondary sources with limited role |
| Counter-evidence | Evidence that complicates the case |
| Overclaim guard | Safer conclusion and prohibited inference |
| Legal/security guardrail | Defamation, CUI/classified inference, operational-detail risk |
| Human-cost guardrail | Rules for dignity, no invented thoughts, no rhetorical exploitation |
| Chapter placement | Proposed chapter or reserve |
| CALM concept | Claim Receipt, Shadow Ledger, Consequence Trail, Learning Escape Route, Second-Use Test, or Reflexivity Defense |
| Required next evidence | What must be found before use |

## Claim-to-Source Row

| Field | Requirement |
| --- | --- |
| Claim ID | Stable identifier |
| Claim text | One sentence |
| Claim type | Factual, analytic, normative, methodological, historical, or illustrative |
| Source ID | Stable source reference |
| Page/section | Page, paragraph, table, figure, or timestamp |
| Quote candidate | Short exact quote or paraphrase note |
| Support strength | Direct, partial, background, contradiction, or no support |
| Boundary | What the source does not prove |
| Countercheck | Source or question that could weaken the claim |

## Human-Cost Method Note

Human-cost cases require extra care. Reconstruct only from public records and
reporting. Do not invent private thoughts. Do not use casualties or failures as
props. The ethical purpose must be to show how institutional representations
reach real people, not to harvest emotion.

## Usefulness Check

This template prevents source laundering. It forces every case to state what it
supports, what it does not support, what counter-evidence exists, and what
guardrails control professional and human risk.

## Claim Boundary

This template does not prove any case. It defines the minimum evidence structure
required before a case can enter Paper Armies, the doctrine manual, or the
toolkit.
