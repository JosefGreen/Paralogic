## Capture Levels

| Level | Meaning | Manuscript Use |
| --- | --- | --- |
| Anchor identified | Public source page or official landing page found | Not enough |
| Public web anchor verified | Source identity and relevance verified from public page | Discovery/support only |
| PDF captured | Source document saved or reliably linked | Still not enough |
| Page extracted | Claims tied to pages/sections/tables | Candidate manuscript support |
| Counterchecked | Limitations and contrary evidence reviewed | Stronger candidate |
| Manuscript-ready | Page-level support, boundaries, counter-evidence, and maturity gate satisfied | Allowed for bounded claim |

## Usefulness Check

This log prevents source discovery from being mistaken for source completion.
It records exactly how far each source has moved toward manuscript readiness.

## Claim Boundary

This log does not validate any book claim. It tracks capture progress and keeps
unfinished sources from being promoted.
