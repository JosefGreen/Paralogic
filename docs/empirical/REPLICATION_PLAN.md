# Replication Plan

Status: EMP0 protocol artifact.

## Replication Package

A replication package must include:

- frozen coding manual;
- dataset schema;
- de-identified coded data where allowed;
- source list or access instructions;
- analysis scripts;
- reliability report;
- limitations and exclusions.

## Independent Replication

An independent team must be able to:

1. sample cases using the stated method;
2. apply the coding manual;
3. reproduce agreement metrics;
4. reproduce any reported descriptive statistics;
5. document deviations.

## Claim Boundary

No result is externally replicated until an independent replication packet has
been completed and reviewed.
