# Pilot Protocol

Status: EMP0 protocol artifact.

## Pilot Goals

- Test whether coders can apply the manual.
- Identify ambiguous code definitions.
- Estimate coding time and disagreement rates.
- Produce no validation claim.

## Minimum Pilot Packet

1. Case selection rationale.
2. Source artifacts.
3. Two independent coders.
4. Adjudication notes.
5. Reliability summary.
6. Rule revision log.
7. Limitations statement.

## Exit Criteria

The pilot exits only with a revised protocol and a list of remaining validity
risks.
