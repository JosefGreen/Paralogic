# Construct Validity Plan

Status: EMP0 protocol artifact.

## Construct Boundaries

Each construct must include:

- formal definition reference;
- empirical indicator;
- exclusion criteria;
- expected failure modes;
- examples and counterexamples.

## Validation Checks

1. Face-validity review by domain experts.
2. Convergent comparison with nearby constructs.
3. Discriminant comparison against prohibited overclaims.
4. Sensitivity analysis for uncertain codes.
5. External replication before any strong empirical claim.

## Boundary Rule

Formal proof status, EFC output, and PYC output cannot substitute for construct
validity.
