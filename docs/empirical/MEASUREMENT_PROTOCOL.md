# Measurement Protocol

Status: EMP0 protocol artifact.

## Measurement Targets

1. ISF components: uses, claims, support degradation, treatment as adequate.
2. M8 components: power relevance, validity dependence, omission.
3. Insight components: evaluator acceptance, transition license,
   contradiction-addressing, defeater absence.
4. Boundary outcomes: harm, responsibility, repair, guilt, and validation are
   measured only through explicit bridge/protocol instruments.

## Required Evidence

Each measurement requires:

- source text or artifact reference;
- coder rationale;
- source span;
- declared context;
- uncertainty note when applicable.

## Output

Measurements populate `DATASET_SCHEMA.json`. The protocol produces coded
observations, not empirical validation by itself.
