# Reliability Plan

Status: EMP0 protocol artifact.

## Procedure

1. Train coders on the coding manual.
2. Double-code an initial calibration set.
3. Compute agreement per code.
4. Revise ambiguous rules.
5. Double-code the pilot set after revisions.
6. Report agreement and unresolved ambiguity.

## Metrics

Primary metric: percent agreement per code.

Planned secondary metrics: Cohen's kappa for two-coder studies and Krippendorff's
alpha for multi-coder studies, once data exists.

## Stop Conditions

Do not claim empirical validation if reliability is untested, below threshold,
or not reported.
