# Roadmap

## Phase 0: Checked Minimal Kernel

Complete enough to build:

- status labels;
- core typed vocabulary;
- one-object semantics;
- ISF, M8, and ValidInsight definitions;
- positive implications;
- countermodel witnesses;
- Lean regression examples.

## Phase 1: Many-Sorted Syntax

Add:

- terms;
- typed variables;
- predicate symbols;
- function symbols;
- formulas;
- well-formedness.

## Phase 2: Model Theory

Add:

- carrier families;
- interpretations;
- assignments;
- satisfaction relation;
- semantic consequence;
- countermodel definition.

## Phase 3: Frame and Context Calculus

Add:

- frame boundaries;
- context embeddings;
- frame morphisms;
- projections;
- translation preservation profiles.

## Phase 4: Evaluator Calculus

Add:

- evaluator criteria;
- acceptance/rejection/abstention;
- soundness relative to criteria;
- incompleteness relative to domain;
- evaluator non-collapse theorems.

## Phase 5: Insight and Delta Calculus

Add:

- delta classifiers;
- false insight;
- null insight;
- collapse;
- non-finality theorems.

## Phase 6: ISFT Expansion

Add:

- Adeq by domain;
- M1-M12 definitions;
- support bases;
- claim bases;
- suppression algebra;
- bounded completeness theorems.

## Phase 7: Normative and Empirical Boundaries

Add:

- harm bridge;
- repair-obligation bridge;
- accountability bridge;
- governance-legitimacy bridge;
- empirical validation protocols;
- reliability and validity artifacts.
